# ecommerce_product_analytics
User behavior analysis, conversion funnel, revenue drivers (PostgreSQL)

## Project Overview
This project analyzes e-commerce user behavior data to understand:
- conversion funnel performance
- revenue contribution by users
- key drivers of purchase conversion
- risk factors such as cart abandonment and returns

## Business Goals
- Improve purchase conversion rate
- Increase average order value (AOV)
- Identify high-value users
- Detect product and UX risk areas

## Tools
- PostgreSQL
- SQL (CTEs, subqueries, window functions)

## Project Structure
- `sql/` – analytical SQL views and metrics
- `data/` – source dataset
- `docs/` – business insights and hypotheses
